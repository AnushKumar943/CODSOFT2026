
# Titanic Survival Prediction Project

This project builds a Machine Learning model using the Titanic dataset
to predict whether a passenger survived or not.

## Steps to Run

1. Install Python (3.8+ recommended)
2. Place Titanic-Dataset.csv inside this project folder
3. Install dependencies:
   pip install -r requirements.txt
4. Run the project:
   python main.py

The trained model will be saved as:
titanic_model.pkl

## Model Used
Random Forest Classifier

## Features Used
- Pclass
- Sex
- Age
- SibSp
- Parch
- Fare
- Embarked
